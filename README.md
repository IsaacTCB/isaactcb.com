# isaactcb.com

A personal static webpage containing relevant links to my socials.
Built with [Astro](https://astro.build/) and [tailwindcss](https://tailwindcss.com/).

## Building

To generate a static copy, run `npm install` and then
`npm run build`. This will bundle all files into `dist/`.
